## 📖 Overview

TalentAI is a modern AI-powered recruitment platform designed to simplify the hiring process for candidates, recruiters, and administrators.

The platform leverages artificial intelligence to automate resume analysis, candidate matching, cover letter generation, and hiring workflows while providing an intuitive and scalable user experience.

# 🚀 Getting Started

## Prerequisites

- Node.js 20+
- MongoDB
- npm
- Git

---

## Installation

### Clone Repository

```bash
git clone https://github.com/leila140/TalentAI.git

cd TalentAI
```

### Client

```bash
cd client

npm install

npm run dev
```

### Server

```bash
cd server

npm install

npm run dev
```

---

## Environment Variables

Create a `.env` file.

```env
MONGODB_URI=

JWT_SECRET=

OPENAI_API_KEY=

CLOUDINARY_NAME=

CLOUDINARY_API_KEY=

CLOUDINARY_SECRET=
```

---

## Docker

```bash
docker compose up --build
```
